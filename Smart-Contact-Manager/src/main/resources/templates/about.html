About Smart Contact Manager

Smart Contact Manager is a web application designed to help users manage their personal contacts efficiently.

The application allows users to:

  • Add new contacts
  • Edit existing contact details
  • Delete contacts
  • Search for contacts by name, phone, or email
  • Manage contacts securely with user authentication and authorization

Key Features

User Authentication

Secure login and registration system for users.

Role-based Access

Different access levels for regular users and admins.

Contact Management

Add, view, edit, and delete contacts.

Search Functionality

Quick and easy search options for finding contacts.

Responsive Design

The application is designed to work well on both desktop and mobile devices.

Technology Stack

  • Spring Boot: For building the backend and handling API requests
  • Spring Security: For securing the application with user authentication and authorization
  • Thymeleaf: A templating engine used for rendering dynamic views
  • Bootstrap: For styling and making the application responsive
  • H2 Database: For storing user data and contact information (you can switch to MySQL or other databases as required)
